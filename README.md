# CMake template for SDL using CPP

A quick template for making a game using CMake and static linked SDL libraries.

## Installation

Install your project with cmake

### Linux and MACOS
```bash
  cd <your project>
  git submodule update --init
  mkdir build
  cd build
  cmake ..
  make
```
